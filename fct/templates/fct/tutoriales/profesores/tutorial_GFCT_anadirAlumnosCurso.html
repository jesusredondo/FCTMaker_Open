Añadir Alumnos en Cursos

Relacionar los Alumnos con los Cursos mediante un Histórico permite que los alumnos...

  • Tengan asociada una Empresa para un curso en concreto. Si no necesita empresa, marcar la empresa con nombre "NO EMPRESA".
  • Puedan entregar propuestas de proyecto para ese curso en concreto.
  • Puedan realizar las entregas finales de proyecto para ese curso en concreto.

Para asociar un alumno a un curso se emplea la Tabla "Histórico_Alumno_Cursos". Crea un nuevo registro de tipo histórico y asocia el Alumno con el Curso para que pueda participar.

Un Histórico_Alumno_Cusos" relaciona un Alumno y un Curso. Para cada histórico se registran:

  • La Empresa en la que el alumno realizará las FCTS. Si no debe tener empresa, o si todavía no sabes cuál es, usa la empresa denominada "NO EMPRESA". Puedes modificar la empresa de un histórico editando el histórico en cualquier momento.
  • Las propuestas de proyecto que ha realizado el alumno para ese histórico. No es necesario rellenarlo. Sólo lo editaremos cuando varios alumnos participen en el mismo proyecto.
  • Las entregas de proyecto que ha realizado un alumno para ese histórico. Se gestionan internamente.
Si el alumno o el curso no están creados, revisa los turoriales para crear un nuevo alumno y crear un nuevo curso.

Recuerda, debes asociar el mismo Alumno con varios Cursos distintos si participa en varios periodos. De esta manera evitamos duplicidad de datos.
